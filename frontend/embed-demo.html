GetMee AI Chatbot - Embeddable Widget

This page demonstrates the embedded GetMee chatbot. Look at the bottom-right corner for the chat bubble.

How to embed on your WordPress site

Add this single script tag before </body> in your WordPress theme or page:

<script
  src="https://your-domain.com/getmee-chatbot.js"
  data-api-base="https://your-backend-api.com"
  data-logo-url="https://your-domain.com/logo.png"
></script>

Configuration attributes

  • data-api-base — URL of your GetMee backend API (default: http://localhost:8001)
  • data-logo-url — URL of the bot avatar/logo image (optional)

Manual initialization (optional)

<script src="https://your-domain.com/getmee-chatbot.js"></script>
<script>
  GetMeeChat.init({
    apiBase: "https://your-backend-api.com",
    logoUrl: "https://your-domain.com/logo.png"
  });
</script>
Note: The chatbot supports English and Spanish with a built-in toggle button in the header.

Features

  • Floating chat bubble (bottom-right corner)
  • English / Spanish language toggle
  • 5 sample quick questions per language
  • Email collection for support escalation
  • Fully self-contained — no CSS conflicts with host site
  • Works on any website (WordPress, Wix, static HTML, etc.)
